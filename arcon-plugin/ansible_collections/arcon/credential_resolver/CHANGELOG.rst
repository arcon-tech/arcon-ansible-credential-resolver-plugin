arcon.credential_resolver Release Notes

v1.0.0
======

Release Summary
---------------

Initial release of the ARCON PAM credential resolver lookup plugin.

New Plugins
-----------

Lookup
~~~~~~

- arcon.credential_resolver.arcon_plugin - Retrieve credentials from ARCON PAM using token-based authentication.
- arcon.credential_resolver.arcon_dv_plugin - Retrieve credentials from ARCON PAM DV using token-based authentication.
